Install in 5 seconds

mkdir -p ~/.claude/skills/barren && curl -sL https://raw.githubusercontent.com/hilberthiggs-hash/data-barrens/main/skill/SKILL.md -o ~/.claude/skills/barren/SKILL.md click to copy

Restart Claude Code, type /barren
